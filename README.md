Includes starter files for HTML, CSS, JS, and image support out of the box.

## ğŸ§° Available Commands

- **`npm run build`**  
  Builds the project using the production configuration (`webpack.prod.js`).

  - Outputs optimized and minified files to the `dist/` folder.

- **`npm start` OR `npm run dev`**  
  Starts the Webpack development server with live reloading.

  - Opens automatically in your browser.
  - Uses `webpack.dev.js` for faster, non-minified builds.

- **`npm run deploy`**  
  Deploys the contents of the `dist/` folder to the `gh-pages` branch on GitHub.

  - Use this after running `npm run build`.

- **`npm run clean`**  
  Deletes the `dist/` folder to start with a fresh build.
  - On Windows, use `rimraf dist` instead of `rm -rf dist` for compatibility.
